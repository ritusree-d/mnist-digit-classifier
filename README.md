# MNIST Handwritten Digit Recognition

A CNN-based handwritten digit classifier built using TensorFlow and Google Colab.

## Features
- Loads MNIST dataset automatically
- Trains CNN model
- Predicts handwritten digits
- Visualizes predictions

## Tech Stack
Python, TensorFlow, NumPy, Matplotlib, Google Colab

## Accuracy
98%+

## How to Run
Open notebook in Google Colab and run all cells.
